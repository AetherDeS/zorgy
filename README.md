<h1 align="center">Zorgy</h1>

### Описание
Zorgy - это гибкая консольная утилита для резервного копирования папок или файлов пользователя таких как /home, /var, /opt и т.д.

#### Поддерживаемый операционные системы:
```
Arch linux
```

### Установка   
Скачать репозиторий через   
```
git clone https://github.com/AetherDeS/zorgy.git && cd zorgy
``` 
скрипт запустится сразу после установки   

вручную можно запустить с помощью   
```
python ./main.py
```
